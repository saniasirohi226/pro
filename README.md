Welcome to **Gerícht**, a premium, beautifully crafted website designed for a high-end restaurant. Combining aesthetic elegance, modern web technologies, and a timeless design philosophy to deliver an immersive and seamless user experience.

---

## ✨ Features That Shine

### 🎨 Immersive & Elegant Design
- **Sophisticated UI**: Visually stunning interface using an elegant color palette inspired by fine dining.
- **Premium Typography**: Integrates Google Fonts (*Great Vibes*, *Cormorant Upright*) for a sophisticated, classic aesthetic.
- **Smooth Animations**: Powered by **GSAP (GreenSock)** & ScrollTrigger, elements fade, slide, and scale gracefully as you scroll down the page, creating a dynamic, living interface.
- **Responsive Layout**: Designed to look breathtaking and function flawlessly across desktops, tablets, and mobile phones.

### 📅 Seamless Reservations
- **Integrated Booking System**: A fully functional reservation form that allows guests to book tables with ease.
- **Automated Data Capture**: Form submissions are integrated seamlessly to a central cloud-based spreadsheet (Google Forms integration).

### 📸 Interactive Photo Gallery
- A custom-built photo gallery with smooth horizontal navigation controls.
- Interactive hover effects to showcase our culinary masterpieces creatively.

## 🛠️ Technology Stack

| Technology | Usage |
|:---:|:---|
| **HTML5** | Core structure and semantic architecture |
| **CSS3** | Vanilla CSS for elegant styling and Flexbox/Grid layouts |
| **JavaScript (ES6+)** | Dynamic interactivity, event handling, and form validation |
| **GSAP & ScrollTrigger** | High-performance, silky-smooth scroll animations |
| **FontAwesome** | Clean, scalable vector icons |
| **Google Forms/Sheets** | Serverless backend for reservation tracking |

## 🚀 Getting Started

Want to run this beautiful project locally? It's simple!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/saniasirohi226/pro.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd pro
   ```
3. **Run the project:**  
   Simply open the `index.html` file in your preferred web browser, or use a tool like **Live Server** in VS Code for a better development experience.
