# 🍽️ Gericht - Fine Dining Experience

> *"The Key To Fine Dining"*

👉 [**Visit Live Website**](https://saniasirohi226.github.io/pro/)

**Gericht** is a premium, beautifully crafted website designed for a high-end restaurant. It combines aesthetic elegance with modern web technologies to deliver an immersive user experience.

---

## ✨ Key Features

### 🎨 Immersive Design
- **Sophisticated UI**: A visually stunning interface using a curated color palette and modern typography (*Great Vibes*, *Cormorant Upright*).
- **Smooth Animations**: Powered by **GSAP** (GreenSock), elements fade, slide, and scale gracefully as you scroll, creating a dynamic feel.
- **Responsive Layout**: Fully optimized for all devices, from desktops to mobile phones.

### 📅 Seamless Reservations
- **Integrated Booking System**: A functional reservation form that allows guests to book tables instantly.
- **Automated Data Handling**: Reservations are automatically captured and organized in a central cloud-based spreadsheet via Google Forms integration.

### 📸 Interactive Gallery
- A custom-built photo gallery with smooth navigation controls to showcase culinary masterpieces.

---

## 📊 Live Reservation Dashboard

---

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Animation**: GSAP (GreenSock Animation Platform) & ScrollTrigger
- **Backend/Data**: Google Forms & Sheets Integration (Serverless architecture)
- **Icons**: FontAwesome
